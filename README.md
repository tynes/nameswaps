# nameswaps

Trustlessly trade Handshake names

This is heavily WIP, do not use

## Usage

### Node

The node runs `hsd` with the `nameswaps` plugin.

```bash
$ npm i
$ ./bin/node
```

### Swaps Wallet CLI

```bash
$ ./bin/swapswallet-cli
```
